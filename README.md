This is a fully interactive digital character sheet for the Stargate SG-1 tabletop RPG, built for the Phoenix Site campaign setting. It runs entirely in your browser as a single HTML file with no apps, no accounts and no installation required. Just open it and play.
It was built by a fan, for the community. Feel free to use it, share it and adapt it for your own campaigns.

## How to use it
Download the HTML file and open it in Chrome, Firefox or Safari on any device. On mobile, open it in your browser and add it to your home screen for easy access so it behaves just like an app.

**Android**: Open in Chrome, tap the three dots menu and select Add to Home Screen
**iPhone / iPad**: Open in Safari, tap Share and select Add to Home Screen
**PC / Mac / Linux**: Open directly in any browser or just bookmark it

## Features

- All 11 playable races with auto-populating traits and abilities
- All 6 classes with auto-populating proficiencies, saving throws and level 1 to 5 class features
- Automatic calculations for HP, Armour Class, Initiative, Passive Senses, Skill modifiers, Proficiency Bonus and Determination Points
- Full armour system where Light armour adds your DEX modifier automatically, Heavy armour uses a fixed AC value, and Head and Held slot bonuses all feed into your total AC
- Character portrait upload that adapts to any image proportions
- Weapons and Attacks table with wrapping text rows
- Death Saves tracker, Exhaustion tracker, Moxie counter, Tension die selector and Inspiration toggle
- Full Background tab with personality traits, ideals, flaws, backstory, allies and organisations and session notes
- Equipment tab with standard base kit checklist, carry load, armour slots and additional inventory
- 16 colour themes, 8 dark and 8 light, each themed around a faction or species from the Stargate universe including SGC, Tok'ra, Goa'uld, Nox, Ancient, Asgard, Keth'alis and Furling
- Auto-saves to your browser on every keystroke so you never lose your work
- Export your character as a JSON file for backup or to move between devices
- Import a previously saved JSON file to restore your character on any device
- Fully responsive layout that adapts to PC, tablet and mobile screen sizes

## Saving your character
The sheet auto-saves to your browser's local storage every time you make a change. For a more permanent backup, or to use your character on a different device, use the Export button in the Options tab to download a small .json file. Keep this somewhere safe like Google Drive or iCloud. To restore it, open the sheet and use the Import button.

## Themes
Open the Options tab to choose from 16 colour themes. Your chosen theme saves with your character file.
